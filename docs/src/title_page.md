# My Rust Playground

_by Ilario A. Azzollini_

As a researcher and an engineer — not a “pure software developer” (at least not by birth) — I see programming languages as tools. In my view, a language is always a means to an end.

Because I work close to the development side of things, I care deeply about developer experience. Rust offers a powerful and well-designed ecosystem. The tools are reliable, consistent, and pleasant to use. They make it easier to write high-quality code and focus on the problem instead of fighting the language.

[This repository](https://github.com/ilarioazzollini/rust_playground) is my Rust playground: a place where I collect what I study, experiment with new ideas, and explore concepts I want to understand more deeply.
